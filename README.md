# wallpaper_project
Stranica za skidanje __wallpapera__ koja sadrzi mnoge kategorije visokokvalitetnih pozadina

### funkcionalnosti:
*kategorije- klikom odvesti  a ekrn sa kategorijom
*download-odabir kvaliteta
*serch-pretraga po nazivu

### kategorije:
*priroda 
*drvo
*lisce
*ptice
*zivotinje u prirodi
